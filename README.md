# 🧠 AI Subject Finder Pro

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Made with HTML/CSS/JS](https://img.shields.io/badge/Made%20with-HTML%2FCSS%2FJS-orange)](https://developer.mozilla.org/)

A **production-ready, self-contained** web application that helps students, researchers, and developers discover **50+ free AI tools** across multiple subjects.

## ✨ Features

- 🔍 **Real-time search** – Filter by tool name, subject, or feature
- 📚 **50+ free AI tools** – Maths, Physics, Programming, Writing, Research, Art/Design, Data Science
- 🌙 **Dark mode toggle** – Persistent preference via localStorage
- ⭐ **Tool ratings** – Community-driven ratings (4.4–4.9)
- 🔗 **Direct external links** – Opens safely in new tabs
- 📱 **Fully responsive** – Works on desktop, tablet, and mobile
- 🚀 **Zero dependencies** – Pure HTML/CSS/JS, no frameworks or external libraries

## 🚀 Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy to any static hosting service (GitHub Pages, Netlify, Vercel)
